# 📝 Java ToDo List App

## 📌 Objective
Build a Java Swing-based desktop ToDo application to add and delete tasks.

## 🛠 Features
- ➕ Add tasks
- ❌ Delete selected tasks
- 📋 View tasks in a list

## 🔧 Technologies Used
- **Language:** Java
- **GUI:** Swing
- **IDE:** IntelliJ / VS Code / Eclipse
